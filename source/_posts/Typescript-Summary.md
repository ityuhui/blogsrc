## 介绍 

Typescript 是微软开发的一款编程语言，是 Javascript 语言的超集，它可以编译成纯 Javascript 语言。

<!-- more -->

## 安装

```shell
npm install -g typescript
```

## 编译

```shell
tsc *.ts
```

## 执行

```shell
node *.js
```

## 语法总结

### 数据结构

* any 
* number
* string 
* boolean
* 数组
* 元组
* 枚举
* void
* null
* undefined
* never

### 变量声明

var [变量名] : [类型];
